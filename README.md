# Hungry Dinosaur

[Play the game on Scratch](https://scratch.mit.edu/projects/1052393772/)

## Overview
Help the hungry dinosaur eat healthy food while avoiding the ladybug. This game is built using Scratch and is both fun and educational for learning basic game logic.

## How to Play
- Use the **Left Arrow** and **Right Arrow** keys to move.  
- Eat healthy food to increase your score.  
- Avoid the ladybug â€” hitting it will decrease your score.

## Win / Lose Conditions
- Reach a score of **30** to win the game.  
- If your score becomes negative, you lose.

## Notes
- This project is a demonstration of Scratch game design.  
- You can view the code directly in Scratch or download the `.sb3` file from this repository to explore.

Enjoy playing and learning!
